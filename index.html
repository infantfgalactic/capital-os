Legal Setup (India)
5-7 days

WHY THIS MATTERS

US clients need invoices. You need GST compliance. Proper legal structure builds trust and avoids tax nightmares.

ENTITY OPTIONS

  • Sole Proprietorship: Simplest. Just GST registration. Good for $0-$50K/year revenue.
  • LLP (Limited Liability Partnership): Better liability protection. Required for bigger clients. 2 partners minimum.
  • Private Limited Company: Overkill unless doing $100K+ in revenue. Skip for now.

RECOMMENDED: START AS SOLE PROP, UPGRADE TO LLP AT $25K REVENUE

SETUP STEPS (SOLE PROP)

  • Register for GST via GST Portal (if annual turnover will exceed ₹20 lakh)
  • Open current account at ICICI/HDFC (required for international payments)
  • Get PAN card (you already have this)
  • Use Zoho Books or Wave for invoicing/accounting (free tiers available)
  • Hire CA for quarterly GST filing (₹2000-3000/quarter)

CONTRACTS & INVOICING

  • Use Bonsai or PandaDoc for client contracts (templates available)
  • Invoice in USD, get paid in USD
  • State services as "IT/Software Consulting" (GST exempt under export of services)
  • Keep FIRA (Foreign Inward Remittance Advice) for all payments

SUCCESS CRITERIA

  • GST registration complete (if required)
  • Current account open and USD-ready
  • Invoice template ready with GST number
  • CA on retainer for compliance

Payment Rails (Wise + Stripe)
2-3 days

WHY THIS MATTERS

Clients won't wire ₹8 lakh to your savings account. You need professional payment infrastructure that works for international B2B.

WISE (FORMERLY TRANSFERWISE)

  • Open Wise Business account (free, 2-day approval)
  • Get USD, EUR, GBP virtual accounts (clients pay locally, you get INR)
  • Fees: 0.5-1% vs 3-5% for bank wire
  • Use for: Wire transfers, ACH, international invoices
  • Verification needed: PAN, GST cert, current account statement

STRIPE (FOR CARD PAYMENTS)

  • Register Stripe account (India-supported since 2021)
  • Accept credit cards for smaller invoices ($500-$2000)
  • Fees: 2.9% + $0.30 per transaction
  • Use Stripe Invoicing (no coding needed)
  • Auto-reminders for unpaid invoices

PAYMENT TERMS STRATEGY

  • Projects under $5K: 50% upfront, 50% on delivery
  • Projects $5K-$20K: 30% upfront, 40% at midpoint, 30% on delivery
  • Retainers: Monthly recurring via Stripe (auto-charge on 1st of month)
  • Net 15 terms (payment due 15 days after invoice)
  • Late fee: 1.5% per month after Net 15

SUCCESS CRITERIA

  • Wise + Stripe accounts live and verified
  • Test transaction completed ($1 deposit)
  • Invoice templates include payment links
  • Average payment time under 10 days

WHY THIS MATTERS

REQUIRED COVERAGE

  • Cyber Liability: Covers data breaches, ransomware. $500K coverage. Cost: ₹30K-50K/year.
  • General Liability: Covers property damage, accidents. $1M coverage. Cost: ₹20K/year.

INSURERS (INDIA)

  • Hiscox (best for tech consultants)
  • ICICI Lombard
  • Bajaj Allianz

LEGAL DOCS

  • MSA (Master Service Agreement): Standard terms for all clients. Get template from LegalZoom or Rocket Lawyer.
  • SOW (Statement of Work): Project-specific scope, deliverables, payment terms.
  • NDA: Mutual non-disclosure for client data.
  • IP Assignment: Client owns all work product (required for most RIAs).

SUCCESS CRITERIA

Security Review (AppExchange Req)
20-30 days

WHY THIS MATTERS

Salesforce AppExchange requires security review for all paid apps. Passing this is non-negotiable for credibility and distribution.

SECURITY REVIEW CHECKLIST

  • Code Quality:
    • All Apex classes must have 75%+ test coverage
    • No hardcoded credentials or API keys
    • SOQL injection prevention (use bind variables)
    • XSS prevention in all Lightning components
    • CRUD/FLS (Field-Level Security) enforcement
  • Data Security:
    • Encryption for sensitive fields (SSN, account numbers)
    • Audit trail for all data access
    • Role-based access controls
    • No data leakage via debug logs
  • Compliance:
    • GDPR compliance (data export/deletion)
    • SOC 2 Type II audit (hire auditor, $10K-15K)
    • Privacy policy + Terms of Service

TOOLS TO USE

  • Checkmarx: Static code analysis (free tier available)
  • Salesforce Code Analyzer: Built-in CLI tool for scanning Apex/LWC
  • PMD: Open-source code quality checker
  • OWASP ZAP: Security testing for web components

SUBMISSION PROCESS

  • Submit app for security review via AppExchange Partner Portal
  • Wait 4-6 weeks for initial feedback
  • Fix all Critical and High issues
  • Re-submit (usually 2-3 rounds needed)
  • Once approved, app goes live on AppExchange

SUCCESS CRITERIA

  • Zero Critical or High security issues
  • AppExchange security badge earned
  • SOC 2 audit passed
  • App listed publicly on AppExchange
